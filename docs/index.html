Minion's Pitwall Program

MPP

Driver가 Room을 만들고 Engineer가 Room에 입장하는 팀형 피트월 워크플로를 제공합니다.

v0.1.0 릴리스 노트 GitHub 저장소
Windows 다운로드 (.exe) 포터블 다운로드 (.exe)

파일이 아직 업로드되지 않은 경우, 최신 릴리스 페이지에서 직접 받으세요.

중요: Source code (zip/tar.gz)는 설치 파일이 아닙니다. Windows 사용자는 반드시 .exe 자산을 다운로드하세요.

Driver Flow

1) frc-pitwall-sender.exe 다운로드/실행

2) 기본값 확인: Telemetry IP 127.0.0.1, UDP Port 20777

3) Relay URL, Room Title, Password 입력

4) Start Session으로 Room 생성

5) 생성된 Room 링크/Password/Permission Code를 엔지니어에게 공유

Engineer Flow

1) pitwall 접속 후 Room Lobby 열기

2) active room list에서 대상 Room 선택

3) Password 입력, 필요 시 Permission Code 입력

4) Join Room으로 입장

5) Room Session 내에서 live/replay/ops/overlay를 문맥 전환

Room 권한 모델

  • Password: 방 입장을 허용하는 기본 키 (문을 여는 열쇠)
  • Permission Code: 전략 제어/노트/고급 도구 권한을 여는 키
  • Password가 없으면 viewer 기본 입장, Password가 있으면 검증 후 입장
  • 동일 Room 협업은 동일 Public Relay endpoint 사용을 전제로 합니다.

주의: 초기 릴리스에서는 Defender/SmartScreen 평판이 낮아 경고가 표시될 수 있습니다.
