핵심 기능

Live Ops Control

세션 상태, health, 공유 정책, 운영 이벤트를 한 화면에서 관리합니다.

Broadcast Overlay

OBS 친화형 오버레이로 랩/포지션/전략/신선도 지표를 깔끔하게 송출합니다.

Replay & Archive

종료 세션 타임라인과 스냅샷을 기반으로 운영 회고를 빠르게 진행합니다.

설치 방법 (Windows)

  1. Windows 다운로드 (.exe) 버튼을 눌러 설치형 또는 포터블 실행 파일을 받습니다.
  2. 다운로드한 파일을 실행합니다.
  3. Windows SmartScreen 경고가 보이면 추가 정보 → 실행을 선택합니다.
  4. 설치형은 설치 마법사 완료 후 실행하고, 포터블은 원하는 폴더에서 바로 실행합니다.

주의: 초기 릴리스에서는 Defender/SmartScreen 평판이 낮아 경고가 표시될 수 있습니다.
